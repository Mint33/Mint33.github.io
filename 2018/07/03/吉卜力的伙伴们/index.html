《吉卜力的伙伴们》

发表于 2018-07-03 | 分类于 读书

《吉卜力的伙伴们》

本书讲什么

铃木敏夫身为吉卜力的制作人，这本书回顾了他负责吉卜力电影的宣发工作中种种，“如何让电影卖座呢”，书中从《风之谷》的宣发开始讲述。

电影宣传原来是这么一回事

这本书让我感受到电影要卖座，宣发的过程中要集合各界的力量。除了传统的打广告，上电视，联合商家，办活动，音乐，乃至制作过程中的拍摄跟拍，后期也可以作为宣传素材在电影播出前放出。

铃木敏夫还独创了“在宣传中投入多少，就会有多少票房收获”的理论。其实我觉得，吉卜力动画之所以卖座，不止是宣传的缘故，人们口口相传从电影中获得的感动也是一个原因。还有就是电影票房与那时的时代背景也息息相关。只不过，龙猫在上映时票房并不高，这倒是让我十分意外。

卖电影真的是为了票房么？

铃木敏夫为了票房似乎有些“不择手段”，但是就像他说得那样，让电影卖出去的目的是为了以后可以一直做动画，毕竟动画电影制作周期之长，成本之高。没有资金的话，是很难支撑一部电影完成的。卖出电影之后就有钱做下去。一直一直做下去，这大概是动画人的期待吧。

也就是说，我的目的不在卖座本身，如果不能回收制作成本，就没法制作下一部电影，所以我才这么拼命宣传。首要任务是要保证能继续做电影。

我的动机可能跟其他制片人不太一样。多数的制片人都是折祷电影能够大卖。而我需要不断考虑的是，如何为高畑先生和宫先生创作好电影提供良好的环境。就像之前反复提到的那样：创作是第一，是否卖座是第二位。

书中还讲述到高畑勋最后一部动画《辉夜姬物语》，这是一部高畑勋特别希望做成的一部动画，制作成本高达50亿日元，可是最后的票房并不如意。

铃木说到这部动画目的并非卖多少票房，让我体会到，原来制作人用心制作的动画有时候并不会成为一部卖座的片子。这其中有时代的变化，也有片子的受众人数的原因。如果真的要做一部卖座的片子，除了本身内容以外，贴合时代和受众广大也很重要啊。

不过，高畑勋仅仅想要做一部自己想做的片子，票房如何，也许并不是他在意的事。

就如铃木敏夫在节目中里说，宫崎骏是娱乐大师而高畑勋是艺术家。

这让我联想到其他创作内容，如果是创作真心想做的，不以讨好别人想做的内容，那么是否会得到市场的认可呢？要获得市场的认可，势必要深谙受众，而又要符合本心，这真的是一件难事。评价者常说，某某人江郎才尽。是否真的是这样呢？叫好又叫座的其中是一份运气。

高畑先生是不追求“炒作出来的高票房”的导演。他认为只有真正理解这部电影的人，才能真心好好享受。这的确是种最理想的状态，我也能理解这种心情。所以《辉夜姬物语》的宣传，我抗拒的只有广告文案。除此以外，打算全部按照高烟先生的意愿来做。

感慨高畑勋与宫崎骏做动画的本心

宫崎骏在成立吉卜力之初，要求要将所有工作人员转为正式，并且工资是固定工资而定。刚开始看到这段，我并未了解这其中的意义，直到后来偶然看到宫崎骏早年来中国访问上海美术制片厂的新闻，交流时候，中方高层只关心吉卜力是如何给工作人员发工资的，并且考虑也将原来的固定工资改为计件——美名其曰“与世界接轨”。宫崎骏对此表示十分失望，本想在动画制作商有所交流。没想到中方关心的仅仅是如何发放工资。

计件工资的对于管理者来说的优势在于，提高产品数量，而对于工作人员来说，这是一种打击创意和创作激情的事情。

宫崎骏在一开始能够那么考虑，其实在位工作人员们考虑，希望他们的辛苦工作能获得应有的回报。

但是我当时正处在逐渐熟悉电影制作的过程中，吉卜力工作室好不容易才走到今天，我希望它维持的时间能长一点。于是我去跟宫先生商量这件事，他提出了三个条件:

第一是要将员工转正。日本的动画工作室多数是召集自由动画人，定好每张画的单价，以按产量付报酬的形式运营吉卜力之前也是如此。但这种运营方式使得动画人的年收入很低，生活也没有保障。

比如说，《魔女宅急便》的制作费是4亿日元。虽然预算比之前的作品增加了许多，但是宫先生对作画水平的要求很高，所以动画人的工作强度也很大。这样的工作密度，员工的平均工资却只有10万日元左右，宫先生和我都觉得很痛心。如果将这些以动画人为主的临时员工转正，支付固定的工资，就要以平均工资翻倍为目标制订计划。

不同的时代背景下，人们娱乐方式发生极大的变化。仅仅是靠宣传，其实已经没有办法获得更多票房了

在书的最后，铃木感慨时代变了。大众消费过度到了数字化的小众兴趣。人们不只是从电影上获得娱乐，电视与互联网的出现，人们对于电影的需求也就越来越小了。感慨铃木敏夫能够有这样的判断。有这样初心的人，真是了不起啊。

像高畑勋或宫崎骏这样的天才，动用大量的人力物力，耗费漫长的周期完成大制作，再经由大规模的宣传问世，期待已久的观众们蜂拥而至，在电影院前排起长队一这样的大众消费时代逐渐终结，不，我觉得已经结束了。

很久以前，只有部分王公贵族可以观赏艺术。有钱人成为赞助人，资助艺术家创作作品。这也为任何人都可以享受的20世纪大众消费社会打下了基础。

在这当中，以胶片的形式出现、可以大量复制的电影，具有胶片所到之处均可同时上映的特点。因此，每位观众只要付极低廉的费用就能观看电影。只要便宜，就能聚集更多的人。就这样，电影成为大众娱乐之王。但是随着电视这种免费的媒体出现后，它的地位就被剥夺了，互联网的出现将它进一步推向深渊。

在这样的时代，花费莫大的预算制作高品质的作品，对制作者来说有很大的风险。于是兴趣开始分流，小众的作品越来越多。音乐是这样，出版也是如此。大制作以及“国民作品”开始从大众娱乐中消亡。

单从电影界来看，大众消费社会也许很久以前就结束了。不过，因为时代处于数字化的过渡时期，我们得益于各种幸运的眷顾，制作了23部作品，其中还诞生了观影人次达2350万、票房收入304亿日元这种出人意料的超高票房作品。

但是，时代变了。

吉卜力本来就是为了制作宫崎骏的作品才成立的工作室，随着宫崎骏的隐退，它的使命自然也就结束了。但是，因为出现了像米林宏昌和宫崎吾朗这样的青年俊才，我有心想为他们留下一个良好的工作环境。
